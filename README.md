# Emotion Detection Chatbot

A simple **text-based emotion detection chatbot** using Python, scikit-learn, and TF-IDF.  
It predicts emotions like joy, sadness, anger, fear, love, and surprise from text and responds interactively.

## How to Use

1. Install dependencies:  
pip install pandas scikit-learn nltk

Run Emotion_chatbot.ipynb and use the interactive chatbot cell.

Dataset
Includes small text files: train.txt, test.txt, and val.txt.
